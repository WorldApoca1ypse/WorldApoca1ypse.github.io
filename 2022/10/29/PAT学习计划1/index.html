3n+1猜想
原题：https://pintia.cn/problem-sets/994805260223102976/exam/problems/994805325918486528
解法：

第一题

#include<stdio.h>
int main()
{
int n, i;
scanf(“%d”, &n);
for (i = 0; n > 1;)
{
if (n % 2 == 0)
n = n / 2;
else
n = (3 * n + 1) / 2;
i++;
}
printf(“%d”, i);
return 0;
}
